# heartin.github.io

Personal résumé / portfolio site for **Heartin J. Kanikathottu** — Author, Architect, Trainer & Tech Evangelist.

Live site: https://heartin.github.io/

## About

A single-page responsive résumé built on the [Start Bootstrap Resume](https://startbootstrap.com/theme/resume) theme (Bootstrap 5). It covers experience, education, skills, publications, interests, awards & certifications, and contact links.

## Structure

- `index.html` — page content (all sections)
- `css/styles.css` — theme styles (includes Bootstrap)
- `js/scripts.js` — scrollspy + responsive navbar behavior
- `assets/img/` — profile images and favicon

## Local preview

It's a static site, so just open `index.html` in a browser, or serve the folder:

```
python -m http.server 8000
```

Then visit http://localhost:8000/.

## Deployment

Hosted with GitHub Pages from the repository root on the default branch.
